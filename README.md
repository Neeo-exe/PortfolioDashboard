\# Portfolio Dashboard



\## Beschreibung



Dieses Projekt entstand im Rahmen des IU Portfolios

„Objektorientierte und funktionale Programmierung mit Python“.



Der Prototyp stellt den Studienfortschritt eines Studierenden dar und berechnet:



\- erreichte ECTS

\- Studienfortschritt in %

\- abgeschlossene Module

\- Notendurchschnitt

\- voraussichtlichen Studienabschluss



\## Voraussetzungen



\- Python 3.13



\## Projekt starten



1\.	Ein Terminal im Projektordner öffnen.

2\.	Das Programm mit folgendem Befehl starten:	-> python main.py
